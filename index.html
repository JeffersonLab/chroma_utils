Some support routines for manipulating files from chroma. Includes "strippers" that extract data (ensemble) files from various Chroma output files. Also contains some fitters.

File extractors

There are "strippers" - codes in chroma_utils/main/strippers - that are used to extract "ensemble" files from xml or binary files generated from chroma

Auxilliary support

There are I/O and Map manipulation routines in chroma_utils/lib/adat

Some fitters

There are some simple fitter codes in main/fitters
